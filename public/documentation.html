Chicken API Documentation

This will be an api for calling Chicken data.

Purpose URL HTTP Method Request data format Response body data format
Get all Chicken Breeds /breeds GET Header with authorized users JWT bearer token A JSON object holding data about all chicken breeds
Get Chicken Breed by name /breeds/[name] GET Header with authorized users JWT bearer token A JSON object holding data about a single breed containing an ID, Name, APA Class, Egg Color, Description, etc.
{
id:1,
breed:Ameraucana Black,
description:description,
eggColor:blue/green,
eggSize:small,
apaClass:AOCCL(All Other Combs Clean Leg),
purpose:eggs,
eggProduction:good,
eggsPerYear:150+,
temperament:docile,
coldTolerance:good,
heatTolerance:good,
varieties: [{
variety:buff,
comb:single,
description:description
}]
}
Get all breeds by egg color /breeds/eggs/[color] GET Header with authorized users JWT bearer token A JSON object holding all breeds with specified color eggs.
Get all breeds by APA Class /breeds/class/[class] GET Header with authorized users JWT bearer token. A JSON object holding all breeds with specified class.
Allow a new user to register /users POST A JSON object holding the new users username, password, and email. Optional: birthday.
{
username:JaneDoe42,
password:small toaster horse leaf2,
email:janedoe@janedoe.com
birthday: mm/dd/yyyy
} 		A JSON object holding the new users information.
Allow user to Login /users POST A JSON object holding "username" and "password" fields. A JSON object holding users information and JWT bearer token.
Allow user to update their user info /users PUT A JSON object holding user data to be updated.
Can send only fields to be update or all fields.
Header with authorized users JWT bearer token.		 A JSON object holding updated user information.
Allow user to view all breeds on their favorites list /users/favorites GET Header with authorized users JWT bearer token. A JSON object with information on all breeds in users favorites list.
Allow user to add breeds to their favorites list /users/favorites/[breedName] POST Header with authorized users JWT bearer token. A JSON object with information on all breeds in users updated favorites list.
Allow users to remove breeds from their favorites list /users/favorites/[breedID] DELETE Header with authorized users JWT bearer token. A JSON object with information on all breeds in users updated favorites list.
Allow user to delete their account /users DELETE Header with authorized users JWT bearer token. A text message indicating that the users account has been deleted.
