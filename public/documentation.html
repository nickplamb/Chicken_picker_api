Chicken API Documentation

An API for pick'n your favorite chicken breeds.

Purpose URL HTTP Method Request data format Response body data format
Get all Chicken Breeds /breeds GET Header with authorized users JWT bearer token. A JSON object with data about all breeds containing an Id, Name, APA Class, Egg Color, Description, etc.
{
id: 1, breed: "Ameraucana Black", description: "description", eggColor: "blue/green", eggSize: "small", apaClass: "AOCCL(All Other Combs Clean Leg)", purpose: "eggs", eggProduction: "good", eggsPerYear: "150+", temperament: "docile", coldTolerance: "good", heatTolerance: "good", varieties: [ { variety: "buff", comb: "single", description: "description" }, ... ] }
Get Chicken Breed by name /breeds/[name] GET Header with authorized users JWT bearer token. A JSON object with data about specified chicken breed.
Get all breeds by egg color /breeds/eggs/[color] GET Header with authorized users JWT bearer token. A JSON object with data about all breeds with specified color eggs.
Get all breeds by APA Class /breeds/class/[class] GET Header with authorized users JWT bearer token. A JSON object with data about all breeds with specified class.
Allow user to Login /login POST A JSON object with "email" and "password" fields. A JSON object with users information and JWT bearer token.
{
user: { favoriteBreeds: [ "606fabac4ecea6d979792c15", ... ], _id: "61ad3f244f35860015640f48", username: "username", password: "salted and hashed password", email: "email address", birthday: "date string", createdAt: "date string of record creation", updatedAt: "date string of last record update", __v: record version (number) }, token: "JSON web token" }
Allow a new user to register /users POST A JSON object with the new users username, password, and email. Optional: birthday.
{
username: "JaneDoe42", password: "small toaster horse leaf2", email: "janedoe@janedoe.com", birthday: "JS date string or mm/dd/yyyy" } 		A JSON object with the new users information.
Allow user to update their user info /users PUT A JSON object holding user data to be updated.
Can send only fields to be update or all fields.
Email address cannot be updated.
Header with authorized users JWT bearer token.		 A JSON object with updated user information.
Allow user to view all breeds on their favorites list /users/favorites GET Header with authorized users JWT bearer token. A JSON object with full records on all breeds in users favorites list.
Allow user to add breeds to their favorites list /users/favorites/[breedId] POST Header with authorized users JWT bearer token. A JSON object with full records on all breeds in users updated favorites list.
Allow users to remove breeds from their favorites list /users/favorites/[breedId] DELETE Header with authorized users JWT bearer token. A JSON object with full records on all breeds in users updated favorites list.
Allow user to delete their account /users DELETE Header with authorized users JWT bearer token. A text message indicating that the users account has been deleted.
